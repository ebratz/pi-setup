# Skill: TanStack Libraries - Visão Geral

## Descrição
Esta skill fornece uma visão geral e acesso a informações detalhadas e exemplos práticos sobre as bibliotecas TanStack, um conjunto de ferramentas JavaScript/TypeScript de alta qualidade para web. Amplamente utilizadas no ecossistema React (e outras frameworks), essas bibliotecas abordam desafios comuns no gerenciamento de estado de servidor (server state), construção de tabelas, gerenciamento de formulários e roteamento.

O objetivo é servir como um guia central para as diversas ferramentas TanStack, direcionando o usuário para o conteúdo específico de cada biblioteca conforme a necessidade.

## O que são as Bibliotecas TanStack?
TanStack é o nome da organização por trás de várias bibliotecas JavaScript/TypeScript agnósticas a frameworks de UI (mas com excelentes integrações para React, Vue, Solid, Svelte). Elas resolvem problemas comuns de desenvolvimento web de forma elegante e eficiente, promovendo performance e manutenibilidade.
